1. Откройте командную строку (терминал)
2. Укажите путь к корневому катологу с проектом: cd "путь к каталогу"
3. Создайте виртуальную среду: conda create -n practice
4. Активируйте среду: conda activate  practice
5. Установите зависимость с помощью команды pip install -r requirements.txt
6. Запустите программу с помощью команды python app.py
